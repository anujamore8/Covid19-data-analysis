# Covid19-data-analysis
# COVID-19 Data Analysis Project

## 📊 Project Overview
This project analyzes global COVID-19 data using Python, Pandas, NumPy, Matplotlib, and Seaborn.

## 🔍 Key Analysis
- Country-wise cases and deaths
- Growth trend analysis
- Death rate comparison
- Most affected countries
- Vaccination trends

## 🛠 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📈 Key Insights
- High correlation between cases and deaths
- Vaccination reduced severity
- Uneven global impact
